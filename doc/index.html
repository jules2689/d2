d2

A system utility tool to provide easy cloning, access, and management for projects. Inspired by the dev tool at Shopify.

WIP.

How to Install?

  1. Add source /path/to/d2/exe/d2.sh to your .bash_profile, .bashrc, etc.

  2. Restart your terminal.

  3. Run d2 help to see what is available.
